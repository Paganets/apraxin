# Планы этажей корпусов

Эта директория содержит изображения планов этажей для различных корпусов торгового комплекса "Апрашка".

## Формат имен файлов

Файлы должны следовать шаблону:
```
building_{код_корпуса}_floor_{номер_этажа}.png
```

### Примеры:
- `building_33_floor_1.png` - Корпус 33, этаж 1
- `building_33_floor_2.png` - Корпус 33, этаж 2
- `building_A_floor_1.png` - Корпус A, этаж 1
- `building_B_floor_3.png` - Корпус B, этаж 3

## Требования к изображениям

- **Формат:** PNG (предпочтительно) или JPG
- **Рекомендуемая ширина:** 800-1200 пикселей
- **Качество:** Достаточное для четкого отображения номеров павильонов
- **Ориентация:** По возможности горизонтальная
- **Цветовая схема:** Светлый фон, четкие границы павильонов

## Как добавить новый план

1. Подготовьте изображение плана этажа в соответствии с требованиями
2. Переименуйте файл согласно шаблону выше
3. Поместите файл в эту директорию
4. План автоматически станет доступен в админ-панели при выборе соответствующего корпуса

## Текущая поддержка

Система поддерживает следующие корпуса:
- **33** - Корпус 33 (5 этажей)
- **A** - Корпус A (5 этажей)
- **B** - Корпус B (5 этажей)
- **C** - Корпус C (5 этажей)
- **D** - Корпус D (5 этажей)

## Примечание для разработчиков

Конфигурация планов находится в файле `/js/floor-plan.service.js` в объекте `FLOOR_PLANS_CONFIG`. Для добавления новых корпусов или изменения конфигурации редактируйте этот объект.
